# restaurant-recommender-cognifyz
content-based restaurant recommendation system for cognifyz internship

## Task 2: Content-Based Recommendation System

### Description
Built a restaurant recommendation system using TF-IDF and Cosine Similarity on a dataset of 20,000+ Indian restaurants.

### Features
- **Cuisine-based recommendations** - Get suggestions for Italian, Chinese, North Indian, etc.
- **Budget filtering** - Filter restaurants under a specific cost
- **Duplicate removal** - No repeated restaurants in results
- **Optimized for performance** - Handles large dataset without RAM issues

### Tech Stack
- Python, Pandas, Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

### How to Run
```bash
pip install pandas scikit-learn
python restaurant_recommender.py
